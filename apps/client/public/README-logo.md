# Logo assets — drop your original image here

Save the original Cooperative Plus logo PNG as:

- `apps/web/public/logo.png`            → used by the `<Logo/>` / `<BrandMark/>` components
- `apps/web/src/app/icon.png`           → Next.js favicon + app icon (auto-detected, no config)
- `apps/web/src/app/apple-icon.png`     → iOS home-screen icon (optional, 180×180)
- `apps/mobile/assets/icon.png`         → Expo app icon + splash (1024×1024 recommended)

Square, transparent or white background. Same file can be reused for all four.
No code change needed once the files are in place.
